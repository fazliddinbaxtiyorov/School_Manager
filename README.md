**Yuklab Olish**
```
git clone https://github.com/fazliddinbaxtiyorov/School_Manager.git
```
**Foydalanish**
```
 pip install -r requirements.txt
```
**Dasturni Ishlatish**
  * Dasturni pasdagi komandani terminal bilan ishga tushuring: 
```
python manage.py runserver
```
